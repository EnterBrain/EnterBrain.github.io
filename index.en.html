About

This script allows you to simplify the gameplay and add new features to the game.
Key features:

  • Minimizing combat interface
  • Spectator Spy
  • Broadcast message
  • Auto-motivator
  • 1Click motivation
  • Demoralizator
  • Links to equipment
  • Full transaction logs
  • Improved market

Preinstall

To use this script, you must have a browser that supports user scripts.
In most browsers, support is provided by means of third-party extensions:

  • Opera: Violentmonkey (but, better: Download Chrome Extension + Tampermonkey)
  • Chrome: Tampermonkey
  • Firefox: Greasemonkey

Install

Now, having a browser that supports custom scripts, we can begin to install it.
Just click on this link and install: Shadow Government Extension
Script for beta test: Shadow Government Extension Beta

Config

By default, not all features are enabled, some functions are disabled.
Settings button, you can find above the button Logout.

Clicking this button will open a popup window with the settings.
